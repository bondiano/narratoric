# Narratoric

A simple game engine for narrative-driven games built with OCaml and Base.

## Quick Start

### Option 1: Automated Setup (Recommended)
```bash
./scripts/setup.sh
```

### Option 2: Manual Setup
1. Create opam switch and install dependencies:
```bash
make setup
```

2. Build the project:
```bash
make build
```

3. Run the example:
```bash
make run
```

## Requirements

- OCaml 5.3.0+
- opam 2.4.1+
